Conociendo sus deseos de adquirir una base de datos, nos resulta muy grato ofrecerle nuestro servicio de bases de datos donde podrá manejar su inventario de productos, base de datos de clientes, nombre de cliente, apellido, dirección, teléfono, ciudad de destino, departamento de destino y métodos de pago asociados a cada venta, se relacionara proveedor a cada producto, precio de compra con este promovedor, costo de envío, impuestos, al precio de compra se le restara todos los gastos asociados y se generara la ganancia obtenida por producto.
